This project's goal is to develop a full working portfolio website readily online that displays the projects that I have done in, but not limited to, COMP4461 Human-Computer Interaction. It is an opportunity to showcase my ability to design and develop a website not only for the use in this course, but also for any future employers.

First, I had to make an initial design using mockup tools to clearly visualise how my project may look like. The professor then introduce to us on design concepts, human psychology in regards to computer interaction, as well as empathising with potential users. Taking these into consideration, I had to modify my design such that it is user-friendly and aesthetically pleasing.

Several changes I have made to the inital design includes:

  • centering components on the index page to not lose users' focus;
  • highlighting buttons that the users' mouse hovers on instead of popping out a triangle pointing to the object to grab users' attention on their current action;
  • use background images for every page so that the pages are not too plain;
  • a "Top" button incase the users ever wanted to return to the index page.
